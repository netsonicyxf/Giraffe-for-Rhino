Giraffe Guide

Abstract

Imports wireframe geometry as structural model into SOFiSTiK.

The Layer Structure

Giraffe organizes structural information using a strict layer structure:

  • all input geometry in a main 'input' layer
  • 1st children layers: 'nodes', 'beams', 'trusses', 'cables', 'springs'
  • 2nd children layers: ' //group number// [ //layer properties// ] { //layer description// }' format
  • 3rd children layers: '{properties}'

example

a layer 2 [ncs 2 ahin mymz] {main facade girders}' within 'beams' layer will produce statement: !#!Label grp 2 - main facade girders beam prop ncs 2 ahin mymz rules: leading and trailing whitespaces allowed group number is mandatory layer properties may be omitted, including square brackets
